## 🎯 主要功能模块

### 游客功能
- 浏览博客首页、详情
- 搜索博客
- 按分类、标签浏览
- 用户注册、登录、找回密码

### 普通用户功能
- **博客管理**: 创作、编辑、删除、发布博客
- **互动交流**: 评论、点赞、收藏、分享
- **社交通信**: 关注、私信、消息通知
- **举报申诉**: 举报内容、提交申诉、提交建议
- **AI智能服务**: AI对话、写作辅助
- **个人中心**: 修改个人信息、头像、密码

### 审核员功能
- **内容审核**: 审核博客、查看审核记录
- **举报处理**: 处理博客、评论、用户举报
- **申诉处理**: 查看和处理用户申诉

### 管理员功能
- **用户管理**: 查看用户列表、封禁/解封用户、管理审核员
- **内容管理**: 管理博客、评论、建议
- **分类标签管理**: 管理分类和标签
- **数据统计**: 查看仪表盘、访问量统计、用户活跃度分析
- **系统管理**: 审核记录查看、消息管理

## 📝 使用说明

1. 每个用例图都是独立的Markdown文件，包含完整的Mermaid图表代码
2. 可以使用支持Mermaid的Markdown查看器或编辑器查看图表
3. 推荐工具:
   - GitHub (原生支持Mermaid渲染)
   - VS Code + Mermaid插件
   - Typora
   - 在线编辑器: https://mermaid.live/

## 🔄 图表关系说明

- `-->` : 表示角色与用例之间的关联
- `-.->` : 表示include或extend关系
- `"继承"` : 表示角色之间的继承关系
